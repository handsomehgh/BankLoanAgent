# 个人贷款业务全流程指南

## 一、贷款申请
- **渠道**：银行网点、手机银行APP、官方网站、合作平台。
  - **操作**：
    1. 客户阅读产品介绍，选择所需贷款类型。
    2. 填写《个人贷款申请表》，包括个人信息、贷款金额、期限、用途等。
    3. 签署《征信授权书》《个人信息处理授权书》。
    4. 提供基础材料：
       - 身份证、户口本或有效居留证件。
       - 婚姻证明（已婚/离异）。
       - 收入证明：近6个月银行流水、工资单、纳税凭证、劳动合同。
       - 资产证明（房产证、行驶证、存款、理财等）。
       - 用途证明（购房合同、装修合同、购车合同、入学通知等）。
       - 经营贷还需：营业执照、经营场所证明、财务报表、购销合同。

## 二、贷款调查
1. **材料审查**：客户经理核验申请材料原件，留存复印件。
   2. **面谈**：与借款人及共同借款人面谈，了解借款用途、还款来源，确认真实意愿。
   3. **实地调查**（适用经营贷）：上门查看经营场所、库存、设备、经营状况。
   4. **征信查询**：经授权后查询个人/企业信用报告。
   5. **还款能力评估**：
      - 计算月收入、月负债总额。
      - DTI=月债务支出÷月收入×100%，一般要求≤55%。
      - 对于自雇人士，收入以可核实的流水、纳税记录为准。

## 三、风险评价与审批
1. **系统初评**：风险模型根据评分卡自动给出建议额度、利率。
   2. **人工复核**：审批人对超出系统权限或特殊情况进行审核。
   3. **审批结论**：批准、有条件批准（需补充材料或调整额度）、否决。
   4. **通知客户**：短信/APP推送/电话通知，批准后进入合同签订。

## 四、合同签订
1. **双录**：对签约过程进行录音录像（根据监管要求）。
   2. **签署文件**：
      - 贷款合同（列明金额、期限、利率、还款计划、违约责任等）。
      - 抵押合同/质押合同/保证合同（如涉及）。
   3. **费用确认**：如涉及评估费、抵押登记费等，明确由谁承担。

## 五、抵质押登记与保险
1. **抵押登记**：房产抵押需到不动产登记中心办理《不动产登记证明》。
   2. **车辆抵押**：在车管所办理抵押登记。
   3. **质押交付**：存单、保单等权利凭证交付银行保管。
   4. **保险**：部分贷款要求购买抵押物财产保险，银行为第一受益人。

## 六、贷款发放
1. **放款条件审核**：确认合同生效、抵押/质押已办理登记、保险已购买等。
   2. **发放方式**：
      - 受托支付：银行按合同约定将贷款直接划至开发商/装修公司/交易对手账户。
      - 自主支付：贷款资金发放至借款人结算账户（一般≤30万元）。
   3. **放款通知**：短信告知客户放款成功，提示还款日和金额。

## 七、贷后管理
1. **还款提醒**：每月还款日前通过短信/APP提醒。
   2. **贷后检查**：
      - 消费贷：抽查大额资金流向，确保未违规流入股市/楼市。
      - 经营贷：定期（至少每半年）上门检查经营、库存、抵押物情况。
   3. **风险预警**：当借款人出现逾期、经营恶化、抵押物贬值等风险信号时，及时预警并采取措施。
   4. **逾期催收**：逾期1-30天短信/电话催收；31-90天加大催收频率、上门催收；超过90天可能委外催收或提起诉讼。

## 八、贷款结清与解押
1. **提前还款**：
   - 客户提前申请，银行计算截至还款日的本息及违约金（若有）。
   - 客户存入足额资金，银行扣款并出具《贷款结清证明》。
   2. **正常结清**：最后一期还款日后，系统自动出具结清记录。
   3. **解除抵押**：持结清证明、身份证等到登记机关办理解除抵押登记，退回房产证。

## 九、异常与特殊处理
- **展期**：因暂时困难申请延长还款期限，需重新评估，可能提高利率。
  - **借新还旧**：发放新贷款偿还原贷款，主要用于经营贷周转。
  - **贷款重组**：调整还款计划、降低利率、免除部分罚息等，需逐级审批。
