# 银行业贷款术语词典 v2.0

> **维护部门**：培训发展部  
> **更新日期**：2025-04-01  
> **说明**：收录与个人贷款业务相关的高频术语，适用于内部培训及客户咨询参考。

---

| 术语 | 英文 | 定义 |
|------|------|------|
| LPR | Loan Prime Rate | 贷款市场报价利率，由18家报价行报价后计算得出的基础利率。银行新发放贷款均以此为定价基准。 |
| BP | Basis Point | 基点，1BP=0.01%。利率报价中常用，如LPR+60BP即LPR+0.60%。 |
| LTV | Loan to Value | 贷款价值比，贷款金额占抵押物价值的百分比。住房贷款首套房LTV一般不超过80%。 |
| 等额本息 | Equal Installment | 每月偿还固定金额，包含本金和利息，月供不变但前期利息占比高。 |
| 等额本金 | Equal Principal Payment | 每月偿还同等本金，利息随本金减少而递减，月供逐月下降，总利息较少。 |
| 气球贷 | Balloon Loan | 贷款期内前期月供较低（可能只还利息），期满一次性偿还大部分本金。 |
| 先息后本 | Interest-Only | 贷款期内每月只偿还利息，到期一次性偿还本金。常用于短期周转。 |
| 提前还款 | Prepayment | 在贷款合同约定的还款计划外，借款人主动提前偿还部分或全部贷款。 |
| 提前还款违约金 | Prepayment Penalty | 因提前还款导致银行利息损失，银行按约定收取的罚金。一般按提前还款本金的1%-3%一次性收取，部分银行满1年后可免收。 |
| 逾期 | Overdue | 未能在约定还款日足额还款。逾期会对征信产生不良记录。 |
| 连三累六 | Consecutive 3 or Accumulated 6 | 征信术语，指连续三个月逾期或累计六次逾期，属于严重信用不良，多数银行会拒绝贷款。 |
| 征信报告 | Credit Report | 由中国人民银行征信中心出具的记录个人信用历史的文件，包含借贷、担保、逾期等信息。 |
| 硬查询 | Hard Inquiry | 因申请贷款、信用卡等由金融机构发起的征信查询，会留下记录并可能影响信用评分。 |
| 负债率 | Debt-to-Income Ratio (DTI) | 每月总债务还款额与月收入之比，是衡量还款能力的重要指标。银行贷款一般要求DTI≤55%。 |
| 抵押 | Mortgage | 以不动产（如房产）作为担保物，借款人违约时银行有权处置并以所得价款优先受偿。 |
| 质押 | Pledge | 以动产或权利（如存单、理财、保单）作为担保物，将凭证交付银行保管。 |
| 保证人 | Guarantor | 承诺在借款人不还款时承担还款责任的第三方，分为一般保证和连带责任保证。 |
| 授信额度 | Credit Line | 银行核定的在一定期限内可循环使用的最高贷款限额。 |
| 循环贷款 | Revolving Credit | 在授信额度和有效期内，可随借随还、反复使用，按实际用款天数和金额计息。 |
| 受托支付 | Entrusted Payment | 银行根据借款人委托将贷款资金直接支付给其交易对手，以保证资金用途合规。 |
| 自主支付 | Self-Payment | 贷款资金发放至借款人账户，由其自行支配使用，银行后续监督用途。 |
| 宽限期 | Grace Period | 贷款发放后的一段时间内，借款人只需偿还利息、不需偿还本金。常见于经营贷初期或留学贷款在学期间。 |
| 展期 | Extension | 延长贷款还款期限，因借款人暂时困难经银行审批同意后执行，可能上浮利率并重新签订合同。 |
| 重组 | Restructuring | 因借款人还款困难调整原贷款条件，可能包括降低利率、延长年限、减少本金或罚息减免等，需逐级报批。 |
| 借新还旧 | Refinancing | 发放新贷款偿还原贷款，主要用于经营贷周转，须严格审核真实资金需求，严禁隐藏不良贷款。 |
| 不良贷款 | Non-Performing Loan (NPL) | 逾期超过90天或虽未逾期但已明显无法偿还的贷款，银行须计提损失准备金。 |
| 贷后检查 | Post-Loan Inspection | 贷款发放后，银行定期对资金用途、借款人信用、抵押物状况等进行跟踪检查，防范风险。 |
| 双录 | Dual Recording | 对贷款签约过程进行录音和录像，作为合规证据留存，保护双方权益。 |
| KYC | Know Your Customer | 了解你的客户，银行业务中需要识别和核实客户身份、评估其风险状况的合规原则。 |
| AML | Anti-Money Laundering | 反洗钱，防止犯罪分子利用银行贷款将非法收入“洗白”的法规和措施。 |
| PII | Personally Identifiable Information | 个人身份信息，包括姓名、身份证号、住址、生物特征等，受到法律严格保护。 |
