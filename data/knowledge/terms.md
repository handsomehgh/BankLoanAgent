# 银行业贷款术语词典

| 术语 | 英文 | 解释 |
|------|------|------|
| LPR | Loan Prime Rate | 贷款市场报价利率，由18家报价行报价计算得出，是银行贷款利率的基准。 |
| LTV | Loan to Value | 贷款价值比，即贷款金额占抵押物价值的比率。住房贷款首套房通常LTV≤70%。 |
| 等额本息 | Equal Installment | 每月偿还同等数额的贷款(包括本金和利息)，月供固定，但前期利息占比高。 |
| 等额本金 | Equal Principal Payment | 每月偿还同等数额的本金，利息随本金减少而递减，月供逐月下降。 |
| 气球贷 | Balloon Loan | 贷款期限内，前期月供较低（可只还利息或少量本金），末期一次性偿还较大本金。常用于经营贷。 |
| 先息后本 | Interest-Only | 每月只还利息，到期一次性还本金。适用于短期周转。 |
| 征信报告 | Credit Report | 记录个人信用历史、借贷记录、逾期情况等的文件，由人民银行征信中心出具。 |
| 逾期 | Overdue | 未在约定还款日足额还款。逾期记录直接影响征信。 |
| 连三累六 | Consecutive 3 or Accumulated 6 | 征信术语，连续三个月逾期或累计六次逾期，属严重信用不良。 |
| 负债率 | Debt-to-Income Ratio (DTI) | 总负债月供与月收入之比，是衡量还款能力的重要指标。 |
| 抵押 | Mortgage | 以不动产（如房产）作为担保，借款人违约时银行有权处置抵押物。 |
| 质押 | Pledge | 以动产或权利（如存单、理财产品）作为担保。 |
| 保证人 | Guarantor | 第三方为借款人提供连带责任担保，借款人违约时承担还款责任。 |
| 授信额度 | Credit Line | 银行核定的可循环使用的最高贷款限额。 |
| 循环贷款 | Revolving Loan | 在授信额度和有效期内，可随借随还，按实际使用金额和天数计息。 |
| 受托支付 | Entrusted Payment | 银行将贷款资金直接支付给借款人的交易对手（如开发商、装修公司），确保专款专用。 |
| 自主支付 | Self-Payment | 贷款资金发放至借款人账户，由借款人自主支配（额度较小消费贷多采用）。 |
| 宽限期 | Grace Period | 贷款发放后一段时间内只需还利息不需要还本金。常见于经营贷。 |
| 提前还款违约金 | Prepayment Penalty | 提前还款时银行收取的费用，通常为提前还款本金的一定比例。 |
| 反洗钱 | Anti-Money Laundering (AML) | 防止通过贷款方式将非法资金合法化的法规与措施。 |
